SAM-Augmented Blending for Enhanced Microplastic Detection Using YOLO11

Published Research Paper | Peer-Reviewed

Developed novel synthetic data augmentation framework addressing underwater microplastic detection using advanced computer vision and deep learning. Leveraged Segment Anything Model (SAM) for automated mask generation, enabling intelligent synthetic dataset creation through blending and copy-paste augmentation techniques. Trained YOLOv11 models on augmented datasets and demonstrated 2.89% improvement in detection accuracy (mAP@0.5:0.95) using SAM + Blended (1BG) approach. Research demonstrates practical effectiveness of synthetic data generation for low-resource environmental monitoring applications. Framework combines state-of-the-art computer vision with real-world environmental challenges, offering scalable solution for automated microplastic detection in oceanographic monitoring systems.

PAM + TDM 4-Servo Real-time Signal Processor

Developed sophisticated MATLAB-based signal processing system for Pulse Amplitude Modulation and Time Division Multiplexing. Implemented real-time signal processing chain demonstrating PAM signal generation with adjustable pulse widths (5-28 microseconds), non-overlapping TDM interleaving across 4 independent channels with 8 kHz master clock, and signal recovery via 4th-order Butterworth low-pass filtering. System computes and visualizes 19 distinct signal outputs including pulse trains, combined TDM waveform, individual channel PAM signals, and recovered signals post-demultiplexing. Achieves smooth 25 FPS real-time animation through vectorized MATLAB operations and pre-allocated graphics handles. Features interactive servo control simulation with synchronized multi-plot visualization demonstrating fundamental signal processing principles in communication systems.
