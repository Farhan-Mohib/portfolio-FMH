### 2. **PAM + TDM 4-Servo MATLAB Simulator**
Real-time educational oscilloscope simulator for pulse amplitude modulation and time division multiplexing.

**Key Features:**
- 8 kHz master clock with 4 independent TDM channels
- Butterworth 4th-order low-pass filter design
- Real-time signal visualization at 25 FPS
- Interactive servo control simulation
- Signal recovery demonstration

**Tech Stack:** MATLAB | Signal Processing | Real-time Graphics | Vectorized Operations

[View Repository](https://github.com/farhan-mohib) | [Simulator Details](https://github.com/farhan-mohib)
